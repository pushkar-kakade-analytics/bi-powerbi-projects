# Financial MIS Dashboard – Dashboard Page Planning

## Overview
The Financial MIS Dashboard is designed as an executive-level reporting solution for monitoring financial, operational, and project performance across the organization.

The dashboard follows a business storytelling approach where each page focuses on a specific analytical objective.

---

# Dashboard Design Principles

The dashboard design follows these principles:

- Executive-friendly layout
- KPI-first reporting
- Minimal visual clutter
- Consistent navigation
- Business-focused storytelling
- Scalable reporting structure

---

# Global Slicers

The following slicers will be available across major dashboard pages:

- Financial Year
- Month
- SBG
- SBU
- BU
- Region
- Project Status
- Project Type
- Project Manager

# Page 1 — Executive Summary

## Objective
Provide leadership teams with a high-level overview of overall business performance.

## Key KPIs
- Revenue Actual
- Revenue AOP
- Revenue Variance %
- Profit
- Profit Margin %
- Net Cashflow
- Accounts Receivable
- Accounts Payable

## Recommended Visuals
- KPI cards
- Revenue trend line chart
- Variance waterfall chart
- Region-wise revenue bar chart
- Project status distribution donut chart

## Business Outcome
Enables quick executive review of financial health and operational performance.

# Visual Design Recommendations

## Theme
- Corporate executive theme
- Minimal color palette
- Clean KPI layout
- High readability focus

## Recommended Color Usage
- Green → Positive performance
- Red → Negative variance
- Blue → Neutral informational metrics
- Orange → Alerts or risks
